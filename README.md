# 🏥 Hospital Management System

This project, **Java-based Hospital Management System**  is designed to streamline healthcare operations such as managing patients, doctors, and appointment scheduling. This project demonstrates practical implementation of **JDBC-based database connectivity** and structured backend logic.

---
##  Project Overview

This system enables efficient handling of hospital workflows by integrating core functionalities like Patient Management, Doctor Records, and Appointment Booking with a relational database.

It is built as a **console-based application** focusing on backend logic, database design, and real-world use cases.

---
##  Key Features

- 👨‍⚕️ **Doctor Management**
    - Add and manage doctor details
    - Track doctor availability
  
- 🧑‍🤝‍🧑 **Patient Management**
    - Register new patients
    - Store and retrieve patient records

- 📅 **Appointment Booking**
    - Schedule appointments between patients and doctors
    - Validate doctor availability

- 🔗 **JDBC Connectivity**
    - Seamless integration between Java application and MySQL database

- 🗄️ **Database Design**
    - Structured schema with relationships between entities

---
## 🛠️ Tech Stack
- **Language:** Java
- **Database:** MySQL
- **Connectivity:** JDBC
- **IDE:** IntelliJ IDEA
